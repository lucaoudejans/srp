Get clubbing

De app Get clubbing heb ik individueel gemaakt, wat onderdeel is van een groter project. Het doel van deze app is om studenten andere studenten offline te ontmoeten. Dit moet worden gestimuleerd door mijn app. Als onderwerp heb ik uitgaan, wat dus betekend dat studenten andere studenten kunnen vinden om samen mee uit te gaan. Get clubbing kan hier erg goed bij helpen.

Dit project is gemaakt met xd waardoor het ook uit te proberen is. Neem gerust een kijkje.
